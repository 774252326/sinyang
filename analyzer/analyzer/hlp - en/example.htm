Example: LAT-Analysis bath sample

1.Start <<YourApp>>, create a new file or open an existing file.

2.Click analysis-analysis setup, open analysis setup dialog.

3.In analysis parameters tab, choose LAT-Analysis bath sample for analysis type and input 1.95 for intercept value.

4.In CV parameters tab, input eight values

5.In solution addition parameters tab, fill in solution addition parameters.

Before filling in, double check the solution addition guidance in analysis parameters tab with your steps.

First step, add VMS 25ml:

Insert one step in solution addition list, step no.1, choose VMS for type, input 25 for vol.(ml)/conc.(ml/L), others leave blank.

Second step, add 1000ml/L standrad suppressor of a volume 0.075ml:

Insert one step in solution addition list, step no.2, choose vol. once addition for type, input 1000 for S conc.(ml/L) and 0.075 for vol.(ml)/conc.(ml/L), others leave blank.

Third step, add 4ml sample sollution:

Insert one step in solution addition list, step no.3, choose vol. once addition for type, input 4 for vol.(ml)/conc.(ml/L), others leave blank.

Fourth step, add 25ml/L standrad accelerator several times, 1ml each time, until noramlized charge go above 0.71:

Insert one step in solution addition list, step no.4, choose vol. interval for type, input 25 for A conc.(ml/L) and 1 for vol.(ml)/conc.(ml/L) and 0.71 for end ratio, others leave blank.

6.Save all settings and close analysis setup dialog by clicking OK.

7.Click analysis-start analysis to start as show below. Operation during a running analysis please refer to run analysis.

8.When solution addition completed, click analysis-compute, it prompt you analysis result as shown below. Accelerator concentration 11.8715ml/L.

Administrator may edit the correction values and click compute to recalculate the result.

9.Close result and save data file and exit.
