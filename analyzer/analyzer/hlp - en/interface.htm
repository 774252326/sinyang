Main Window

From top to bottom, there are a title bar, a menu bar, a toolbar, left view, right view, analysis progress window and status bar.

title bar(brown) display software name and opened file name
menu bar(red) main menu
toolbar(orange) buttons of the basic commmands of file, analysis and view menu.
left view(yellow) display experimental data curve, voltammogram
right view(green) display test curve and result(e.g. fitting line, interpolation)
analysis progress window(purple) display analysis progress and charge result.
status bar(blue) software status and command tips
