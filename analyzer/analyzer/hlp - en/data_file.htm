Data File Format

extention of data file is ghb, data file contains all analysis setup parameters, also experimental data.

use new to create a new data file, or use open to open exist data file.

there are four categories in analysis setup parameters: analysis parameters, CV parameters, solution addition parameters and formula corrections. first three categories can be edited by analysis setup command, while formula corrections is editable for administrator only in compute dialog.

new ghb file contains no experimental data, you can only run analysis to add/change experimental data. experimental data curve display in left view.
