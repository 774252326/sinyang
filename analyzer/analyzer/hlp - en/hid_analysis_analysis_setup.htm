Analysis Setup command (Analysis menu)

Open analysis setup window with an 'analysis parameters' tab, a 'CV parameters' tab and a 'solution addition patameteers' tab.

'Analysis parameters' tab:

Analysis type choose analysis method, corresponding solution addition tips will show in the text box below.
Evaluation ratio range in 0~1 for result computation
Calibration curve/Calibration factor/Intercept value enter the value of calibration factor/intercept value or the file path of calibration curve data file.

'CV parameters' tab:

Parameters for charge computation.

Low E limit/High E limit upper limit and lower limit of the x value of voltammogram
Scan rate change rate of scan voltage
Rotation rate rotation rate of electrode
Variation tolerance scan stop when the difference ratio of last charge to second last charge go below variation tolerance.
No. of cycle maximum number of scanning, input 0=infinity.
End integration E/Start integration E upper and lower limit of integration for charge computation, software will find zero point as the start integration E when you select auto.

You must set all the parameters above before you start your analysis process.

'solution addition patameteers' tab:

Fill your solution addition steps in the form:

No. no. of step
TYPE five addition type provided, see below.

Solution addition parameters settings:

TYPE S Conc.(ml/L) A Conc.(ml/L) L Conc.(ml/L) Vol.(ml)/Conc.(ml/L) End Ratio
Vol. once addition standrad suppressor concentration standrad accelerator concentration standrad leveler concentration solution volume to add not use
Vol. interval normalized charge, this step ends when its normalized charge go below/above(according to analysis method in use) specify value.
Conc. once addition S/A/L concentration in plating cell after adding solution not use
Conc. interval S/A/L concentration increment in the plating cell after adding solution normalized charge, this step ends when its normalized charge go below/above(according to analysis method in use) specify value.
VMS not use not use not use solution volume to add not use

Remark:

1.not use: parameter will not be used in computation, you may enter any value.

2.S Conc.(ml/L), A Conc.(ml/L) and L Conc.(ml/L) all equal to 0 is an indicator of an unknown solution, or the sample solution to analysis.

3.When we choose Conc. once addition or Conc. interval, the solution to add must be single compound, or in other words, there is one and only one not be zero between S Conc.(ml/L), A Conc.(ml/L) and L Conc.(ml/L). the value of Vol.(ml)/Conc.(ml/L) is the corresponding compound concentration/concentration increment in the plating cell.

4.For any blank cell in the form, software will automatically set it to a default value, default value of type is Vol. once addition, and of others are 0.
