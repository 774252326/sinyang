Run Analysis

After complete all settings by analysis setup, click start analysis to run analysis. Software then add solution and measure charge according to your settings.

It pauses before each solution addition and prompt volume of solution to add as shown below. It will continue running until resume is clicked.

You may pause the running analysis and it propmt you pause as show below.

Click analysis setup to change solution addition task list during analysis running:

Neither Analysis parameters tab and CV parameters tab of analysis setup dialog is in use, and a solution addition task list shown in Solution addition parameters tab, that is the solution addition setups that it is going to complete.

It will replace the original solution addition task list with a new one at the time when next solution addition step starts after you click OK.

It can also check each solution addition steps according to analysis method in use, if error occurs, it will stop analysis and prompt error message. You need to correct your settings and restart again.

It prompt complete when all solution addition steps completed.

You may click resume to end, or click analysis setup to add new steps.
