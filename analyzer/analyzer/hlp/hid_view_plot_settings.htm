Plot Settings command (View menu)

Open plot setting window with a 'background+legend' tab and a 'curve' tab.

Change the properties of background, legend and curve here.

Background+Legend Tab:

Background Properties£º

Color Style Background color, three options: standrad/light/dark
grid line type grid line type, choose one from five line type or hide grid line.
label font size label font size, range 0~50.
metric font size metric font size, range 0~50.

Legend properties:

show legend show legend when checked.
fix Legend location from the left top corner of plot is fixed when check
alignment options to align legend to four corner of plot or free
size legend size, three option: manual/fix size/auto adjust

legend size setting:

manual enter font size and line length
fix size enter width and height of rectangle in pixel uint to compute the font size, so that legend size will be within the rectangle.
auto adjust auto adjust the width and height of rectangle according to plot size
